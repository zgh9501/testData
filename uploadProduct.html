将一个项目上传到github上，并得到一个网址能访问已发布的项目

创建项目秘钥

a) 项目文件例如：test文件夹——>打开文件夹；

b) 鼠标右键——>调出gitBash；

c) 生成秘钥——>输入ssh-keygen -t rsa（回车）,出现其他的命令，则一直按回车，直到出现2048；

d) 复制秘钥——>之后打开路径，C盘/用户/用户名/.ssh文件夹，打开id_rsa.pub，对其中的内容全选并进行复制；

github上粘贴秘钥，与本地数据进行远程连接

a) 打开github官网，进入settings，左边选项列表里点击SSH and GPG keys，在右边出现一些内容，点击绿色按钮（New SSH key）；

title——>写自己项目名字（英文）
key——>将id_rsa.pub文件中的内容粘贴到内容框里,再点击add SSH key

b) 点击Your profile——>repositories——>new(新建)

repository name——>项目名(test)
description——>项目描述（可以为中文），其余默认

c) 点击Create repository

将项目上传

(1) git init——>回车后有一个.git文件夹

(2) git add .——>将项目文件添加到远程git上

(3) git commit -m “first commit”——>提交的日志描述

(4) git remote add origin git@github.com:(github的用户名)/test.git (项目名)

(5) git push -u origin master——>回车

如果出现yes/no?输入 yes，如果没有，就不用管了
出现master——>master，就表示成功了

通过外网访问自己的项目--创建github的分支

(1) git branch gh-pages——>回车

(2) git checkout gh-pages——>回车

(3) git push -u origin gh-pages——>回车

(4) 出现gh-pages——>gh-pages，就算成功了

(5) 在浏览器打开自己项目网址：用户名.githut.io/项目名

例如：zgh9501.github.io/test
注意: ajax请求的网址必须以https开头，因为github是以https请求为主的
